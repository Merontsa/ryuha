# 流派｜Ryuha

**完成の手前に、未来の芯がある。**

まだ価値と呼ばれていないものに、名前と作法を与える。

---

## これは何か

「流派」は、完成された作品集ではありません。
価値が立ち上がる手前を記録する、制作ノートです。

SNSのnoteではなく、GitHub時代の新しい制作ノートとして構想しました。
privateな思考母艦と、publicに出すHPの中間にある構造──
クローズドにしてオープンな思想ノートです。

## 三つの流派

- **手前派（Temae-ha）** ── 完成前・名づけ前・市場化前の価値を見る流派
- **多想派（Taso-ha）** ── ひとつの素材から多層の意味を読み取る流派
- **棘派（Toge-ha）** ── 痛み・身体性・簡単に消費されない価値を象徴する流派

## 紅花

紅花は、この流派の最初の実践素材です。
単なる地域資源・伝統作物・染色素材ではなく、AI時代における深さ・手仕事・身体性・土地の記憶を考えるための入口として選びました。

## 技術構成

- Google Fonts（Noto Serif JP, Inter）
- GitHub Pages / Vercel でのデプロイに対応
